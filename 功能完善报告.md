## ⚠️ 注意事项

### WebSocket 后端要求

简化版后端（`simple_server.py`）暂不支持 WebSocket，需要添加：

```python
# 后端需要实现
@app.websocket("/ws")
async def websocket_endpoint(websocket: WebSocket):
    await websocket.accept()
    # 推送训练状态更新
    await websocket.send_json({
        "type": "task_update",
        "data": {...}
    })
```

### 图表数据

当前使用模拟数据，实际使用时需要从后端获取：
```typescript
// 替换为真实 API 调用
const rewardData = await trainingApi.getMetrics(experimentId)
```

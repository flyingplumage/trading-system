50+ 高质量组件，覆盖所有场景

完整的类型定义，开发体验好

## ✨ 与 Ant Design 版本的区别

| 特性 | Ant Design 版 | Naive UI 版 |
|------|--------------|------------|
| 框架 | React | Vue 3 |
| UI 库 | Ant Design 5 | Naive UI 2 |
| 风格 | 企业级 | 简洁现代 |
| 主题 | 支持 | 内置明暗 |
| TS 支持 | 好 | 优秀 |
| 体积 | 较大 | 较小 |

**Naive UI 优势：**
- 更简洁的设计
- 更好的 TS 支持
- 更小的打包体积
- 内置主题切换
